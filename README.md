# KISS

KISS는 두 대의 Mac 사이에서 키보드, 마우스, 트랙패드 입력을 공유하는 메뉴바 앱입니다.

서버 Mac의 화면 가장자리로 커서를 이동하면 클라이언트 Mac으로 제어가 넘어가고, 클라이언트 Mac의 반대쪽 가장자리로 커서를 이동하면 다시 서버 Mac으로 돌아옵니다.

## 주요 기능

- 두 대의 Mac 사이에서 키보드와 마우스 공유
- 트랙패드 스크롤과 제스처 전달
- 마우스 더블클릭과 휠 입력 지원
- 클립보드 텍스트 동기화
- 서버와 클라이언트의 한/영 입력 상태 동기화
- 로그인 시 자동 실행 옵션
- 잠자기 또는 노트북 뚜껑 닫힘 시 연결 자동 해제
- UDP 마우스 이동 가속 모드 지원

## 처음 실행할 때 필요한 권한

KISS는 입력을 감지하고 다른 Mac에 전달해야 하므로 macOS 권한이 필요합니다.

처음 실행하면 필요한 권한이 있는지 확인합니다.

1. 손쉬운 사용
2. 입력 모니터링

권한이 없으면 안내 창이 표시됩니다. 안내에 따라 시스템 설정을 열고 KISS를 허용한 뒤 앱을 다시 실행합니다.

권한 설정 위치:

```text
시스템 설정 > 개인정보 보호 및 보안 > 손쉬운 사용
시스템 설정 > 개인정보 보호 및 보안 > 입력 모니터링
```

## 사용 방법

1. 두 Mac에서 KISS를 실행합니다.
2. 입력을 공유할 Mac에서 `서버 (입력 공유)`를 선택합니다.
3. 입력을 받을 Mac에서 `클라이언트 (입력 수신)`을 선택합니다.
4. 서버 Mac에서 클라이언트 Mac의 IP 주소와 비밀번호를 입력해 연결합니다.
5. 서버 Mac의 실제 화면 끝으로 커서를 이동하면 클라이언트 Mac을 조작할 수 있습니다.
6. 클라이언트 Mac의 반대쪽 화면 끝으로 커서를 이동하면 서버 Mac으로 돌아옵니다.

## 연결 상태

연결되면 메뉴바에 상대 Mac의 IP 주소가 표시됩니다.

마우스 이동에 UDP 가속 모드가 사용 중이면 IP 주소 옆에 `+ UDP`가 표시됩니다.

## 연결 끊기

메뉴바에서 `연결 끊기`를 선택하면 현재 연결이 종료됩니다.

클라이언트에서 연결을 끊은 경우 서버도 자동 재연결을 시도하지 않습니다. Mac이 잠자기에 들어가거나 노트북 뚜껑을 닫는 경우에도 연결은 자동으로 정리됩니다.

## 로그인 시 자동 실행

메뉴바에서 로그인 시 자동 실행 옵션을 켜면 Mac에 로그인할 때 KISS가 자동으로 실행됩니다.

## 한/영 입력

서버 Mac에서 한/영 입력 상태가 바뀌면 클라이언트 Mac도 같은 입력 상태를 따르도록 동기화됩니다.

클라이언트 Mac에도 한국어 입력 소스가 추가되어 있어야 한/영 동기화가 정상적으로 동작합니다.

## 문제 해결

입력이 전달되지 않는 경우:

- 두 Mac에서 KISS가 실행 중인지 확인합니다.
- 두 Mac이 같은 네트워크에 있는지 확인합니다.
- 손쉬운 사용과 입력 모니터링 권한이 허용되어 있는지 확인합니다.
- 권한을 새로 허용한 뒤에는 KISS를 다시 실행합니다.

연결은 되었지만 커서가 넘어가지 않는 경우:

- 서버 Mac에서 실제 화면의 좌우 끝으로 커서를 이동했는지 확인합니다.
- macOS 유니버설 컨트롤 또는 다른 화면 배치가 같은 가장자리를 사용 중인지 확인합니다.

한/영 전환이 맞지 않는 경우:

- 클라이언트 Mac에 한국어 입력 소스가 추가되어 있는지 확인합니다.
- 두 Mac 모두에서 입력 소스 설정이 정상적으로 동작하는지 확인합니다.
